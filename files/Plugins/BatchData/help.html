The Batches

Batches enable the GoldenGATE editor to process a list of files automatically, e.g. as an overnight job.

The Batches Menu

The Batches menu provides access to functions for managing and executing Batches:
  • Create: Create a new Batch by selecting a processor, a set of files, and a storage destination.
  • Edit: Open the Batch management dialog.
  • Run: Select a Batch from a list and execute it.

The Edit Batches Dialog

The dialog for Batch management is accessible through the Edit function in the Batch menu. On the right of the dialog, a list displays all existing Batches. In the middle, a set of buttons allows changing the parameters of the Batch currently selected in that list:
  • "Use ..." Buttons: For every type of document processor accessible to GoldenGATE, a button exists which opens a dialog for selecting the processor from a list displaying all avalilable processors of the particular type.
  • Processor: Below the buttons, a label displays the type and name of the processor currently selected.
  • Selected Document Writer: The Document Writer to use for storing the documents after they are processed. The button to the right of the label opens a selector dialog.
  • Storage Destination: The button below displays the folder to store documents to after they are processed. Clicking the button opens a dialog for selecting the folder.
  • File Table: The table displays the files the Batch handels. The right column provides the status of each file.
  • Add File(s): Open a dialog for selecting files. Selecting a folder will add all files contained in that particular folder (excluding sub folders).
  • Remove File(s): Remove the files selected in the table above the button.
At the top of the dialog, a line of buttons provides the functionality for managing the Batches:
  • Create: Open a sub-dialog for creating a new Batch by selecting a source of Annotations and adding the required parameters and the type for the Annotations.
  • Clone: Clone the Batch currently selected. This is pretty much the same as creating a new one, beside the fact that the parameter values are copied.
  • Delete: Delete the Batch currently selected.
  • Run: Execute the Batch currently selected.

The Run Batch Dialog

Clicking the Run button in the Batch management dialog will open a Run dialog for the Batch currently selected. This dialog allows controlling the Batch during execution.
  • File Label: At the top of the dialog, a label displays the file currently being processed.
  • Progress Bar: The progress bar visualizes the share of the files already processed.
  • Interactive: Checking this box allows the processor to interact with the user if necessary, unchecking it forces the processor to work fully automatically.
  • Start Batch: Start executing the Batch (clickable when the Batch is stopped or paused).
  • Pause Batch: Pause executing the Batch (clickable when the Batch is running).
  • Stop Batch: Stop executing the Batch (clickable when the Batch is running).
  • Reset batch: Reset the Batch so processing starts with the first file in the list (clickable when the Batch is stopped).
