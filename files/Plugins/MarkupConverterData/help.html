The Markup Converters

Markup Converters are document processors capable of converting and filtering the XML markup of documents.

The Markup Converters Menu

The Markup Converters menu provides access to functions for managing and applying Markup Converters, and access to a set of functions for direct markup conversion:
  • Select Annotations: Display Annotations of a particular type to individually select them for removing or deleting.
  • Transform Annotations: Transform Annotations of a particular type to Document Parts.
  • Rename Annotations: Rename Annotations of a particular type to a new type.
  • Remove Annotations: Remove Annotations of a particular type from the document.
  • Remove Duplicate Annotations: Remove duplicate Annotations. Two Annotations are considered equal if they have the same type and mark the same part of the document text. The attributes of duplicates will be unioned.
  • Remove Self-Contained Annotations: Remove Annotations contained in another Annotation of the same type.
  • Remove Self-Containing Annotations: Remove Annotations containing another Annotation of the same type.
  • Remove Annotations Contained in ...: Remove Annotations of a particular type if they are contained in another Annotation of another particular type.
  • Remove Annotations Containing : Remove Annotations of a particular type if they are containing an Annotation of another particular type.
  • Remove Annotations Not Contained in ...: Remove Annotations of a particular type if they are not contained in another Annotation of another particular type.
  • Remove Annotations Not Containing : Remove Annotations of a particular type if they are not containing an Annotation of another particular type.
  • Delete Annotations: Remove Annotations of a particular type from the document, deleting the words they mark. This is useful for removing scripts from documents that originate from HTML pages.
  • Rename Annotation Attribute: Rename an attribute of Annotations (of a particular type or of all Annotations).
  • Remove Annotation Attribute: Remove an attribute from Annotations (of a particular type or of all Annotations).
  • Create: Create a new Markup Converter by specifying some mappings.
  • Edit: Open the Markup Converter management dialog.
  • Apply: Select a Markup Converter from a list and apply it to the document currently selected. This function is also available through the Apply Markup Converter option in the Tools menu.

The Edit Markup Converters Dialog

The dialog for Markup Converter management is accessible through the Edit function in the Markup Converters menu. On the right of the dialog, a list displays all existing Markup Converters. In the middle, a table displays the mappings done by the Markup Converter currently selected in that list, where the element name in the "Mapped Tag" column is mapped to the element name in the "Mapping Target" column. Below the table, there are two buttons managing the mappings, and to the left of it, two further buttons allow manipulating the order in which the mappings are applied:
  • Add Mapping: Add a new mapping to the table. This will prompt for a "Mapped Tag", the "Mapping Target" can be entered in the table afterward.
  • Remove Mapping: Remove the mapping currently selected in the table.
  • Up: Move the mapping currently selected in the table up by one row.
  • Down: Move the mapping currently selected in the table down by one row.
At the top of the dialog, a line of buttons provides the functionality for managing the Markup Converters:
  • Create: Open a sub-dialog for creating a new Markup Converter.
  • Clone: Clone the Markup Converter currently selected. This is pretty much the same as creating a new one, beside the fact that the mappings are copied.
  • Delete: Delete the Markup Converter currently selected.

Special Mappings

In order to keep them simple, Markup Converters exclusively consist of tag mapping. For accessing the other functions, like filtering, they provide some special syntax and prefixes for the mapped tags and the mapping targets, which trigger these extended functions:
  • Transform Annotation: Mapping a tag name to "#T" (for Transform) will transform the Annotations of that particular type to Document Parts.
  • Rename Annotations: Mapping a tag name to "#N <newName>" (for reName) will rename the Annotations of that particular type to <newName>. If <newName> is empty, the Annotations will be removed. Since renaming is the default operation, the "#N" is optional.
  • Remove Annotations: Mapping a tag name to "#R" (for Remove) will remove the Annotations of that particular type from the document. If the tag name is empty or "*", all Annotations will be removed, regardless of their type.
  • Remove Duplicate Annotations: Mapping a tag name to "#R-D" (for Remove Duplicates) will remove the duplicate Annotations of that particular type from the document.
  • Remove Self-Contained Annotations: Mapping a tag name to "#R-I" (for Remove Inner) will remove Annotations of that particular type if they are contained in another Annotation of the same type.
  • Remove Self-Containing Annotations: Mapping a tag name to "#R-O" (for Remove Outer) will remove Annotations of that particular type if they contain another Annotation of the same type.
  • Remove Annotations Contained in ...: Mapping a tag name to "#R-CI <outerType>" (for Remove Contained In) will remove Annotations of that particular type if they are contained in an Annotation of type <outerType>. If <outerType> is empty, the mapping will do nothing.
  • Remove Annotations Containing : Mapping a tag name to "#R-CA <innerType>" (for Remove Containing Annotation) will remove Annotations of that particular type if they contain an Annotation of type <innerType>. If <innerType> is empty, the mapping will do nothing.
  • Remove Annotations Not Contained in ...: Mapping a tag name to "#R-NCI <outerType>" (for Remove Not Contained In) will remove Annotations of that particular type if they are not contained in an Annotation of type <outerType>. If <outerType> is empty, the mapping will do nothing.
  • Remove Annotations Not Containing : Mapping a tag name to "#R-NCA <innerType>" (for Remove Not Containing Annotation) will remove Annotations of that particular type if they do not contain an Annotation of type <innerType>. If <innerType> is empty, the mapping will do nothing.
  • Delete Annotations: Mapping a tag name to "#D" (for Delete) will remove the Annotations of that particular type from the document, deleting the text they enclosed.
  • Rename Annotation Attribute: Mapping "<type>.<attributeName>" to "#NA <newAttribute>" (for reName Attribute) will rename the attribute <attributeName> to <newAttribute> for all Annotations of type <type>, while the Annoations itself will not be changed. If <type> is empty or "*", the attribute will be renamed in all Annotations, regardless of their type. If <newAttribute> is empty, the attribute will be removed. Since remaing is the default operation, the "#NA" is optional.
  • Remove Annotation Attribute: Mapping "<type>.<attributeName>" to "#RA" (for Remove Attribute) will remove the attribute <attributeName> from all Annotations of type <type>, while the Annoations itself will not be changed. If <type> is empty or "*", the attribute will be renamed in all Annotations, regardless of their type. If <attributeName> is empty or equal to "*", all attributes will be removed, regardless of their name.

The table below provides a brief summary of the special mappings. <type> retresents any particular Annotation type. If <type> is empty or "*", the operation affects all Annotations, regardless of their type. <attribute> retresents any particular attribute name. If <attribute> is empty or "*", the operation affects all attributes, regardless of their name.

Transform Annotations of type <type> to Document Part <type> #T
Rename Annotations of type <type> to <newType> <type> #N <newType>

Remove duplicate Annotations of type <type> <type> #R-D
Remove self-contained Annotations of type <type> <type> #R-I
Remove self-containing Annotations of type <type> <type> #R-O
Remove Annotations of type <type> contained in Annotations of type <containingType> <type> #R-CI <containingType>
Remove Annotations of type <type> containing Annotations of type <containedType> <type> #R-CA <containedType>
